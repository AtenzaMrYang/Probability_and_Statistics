# 《概率论与数理统计》习题解答

> 存放福建师范大学非数学专业的《概率论与数理统计》课后习题

## 使用方法

1. 推荐使用 `XeLaTeX` 编译主文件，这能在最大程度上避免中文生僻字乱码
2. 项目主文件 `概率论与数理统计.tex` 的第23行，有条被注释的指令
   - 当`\excludecomment{solution}`保持注释状态，编译输出的PDF将包含题目和解答
   - 当`\excludecomment{solution}`取消注释状态，编译输出的PDF只有题目不含解答
3. 关于字体问题，`\documentclass{ctexart}`在 Windows 平台默认使用中易字体（宋体/黑体/楷体），在 macOS 或 Linux 上编译会因缺少这些字体而回退到其他字体，视觉效果会有些许不同，请您使用时留意

## 文件树结构
